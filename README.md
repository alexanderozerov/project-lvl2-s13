###Installation
```$ make install```
```$ npm install p-lvl2-gendiff```
###Usage
```$ gendiff [options] <first_config> <second_config>```
###Options
* ```-h, --help           output usage information```
* ```-V, --version        output the version number```
* ```-f, --format [type]  Output format [pretty, json, plain]```
